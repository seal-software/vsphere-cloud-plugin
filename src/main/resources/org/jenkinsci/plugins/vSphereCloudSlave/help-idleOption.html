What should be done when the agent disconnects.
  • Shutdown: Shuts the virtual machine down, first attempting a Guest shutdown if VMware tools are installed, then a hard power off if VMware tools is not installed or the Guest shutdown fails.
  • Revert: Shuts down the virtual machine like Shutdown, then reverts. If a snapshot name is specified, the revert will be to that snapshot.
  • Revert and Restart: Same behavior as Revert, followed by a shutdown if the machine is on, then a power on.
  • Revert and Reset: Same behavior as Revert, followed by a reset.
  • Suspend: Suspends the virtual machine.
  • Reset: The VM is reset.
  • Reconnect and Revert: The VM is reconnected to Jenkins. If a snapshot name is specified, the VM will be reverted to it when starting up.
  • Nothing: Nothing is done to the VM; it is left untouched.
